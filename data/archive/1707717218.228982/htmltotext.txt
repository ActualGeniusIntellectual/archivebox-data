API users must use this new subdomain to retrieve data. What is it? The data API provides programmatic access to BEA published economic statistics using  industry-standard methods and procedures. BEA's data API includes methods for retrieving a subset of our  statistical data and the meta-data that describes it. The data API and its documentation are  for programmers who are familiar  with the concepts and techniques of retrieving data from Web Services.
